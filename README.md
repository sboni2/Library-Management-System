# 📚 Library Management System

## 🔹 Overview

This project is a **Library Management System** developed using **Spring Boot (Backend)** and **HTML, CSS, JavaScript (Frontend)**.
It allows users to manage books, memberships, and transactions efficiently with role-based access.

---

## 🔹 Features

### 🔐 Authentication

* Admin Login
* User Login
* Role-based access control

### 🛠️ Maintenance Module (Admin Only)

* Add Membership
* Update Membership
* Add Books / Movies
* Update Books / Movies
* User Management

### 🔄 Transactions Module

* Issue Book
* Return Book
* Pay Fine

### 📊 Reports Module

* Master List of Books
* Master List of Movies
* Membership Reports
* Active Issues
* Overdue Returns
* Pending Requests

---

## 🔹 Tech Stack

* **Backend:** Spring Boot (Java)
* **Frontend:** HTML, CSS, JavaScript
* **Tools:** VS Code, Maven, Git, GitHub

## 🔹 How to Run the Project

1. Clone the repository:
   git clone https://github.com/your-username/Library-Management-System.git

2. Navigate to project folder:
   cd Library-Management-System

3. Run the application:
   mvn spring-boot:run

4. Open browser:
   http://localhost:8081/home.html
